### 🌟 Stages of Life Logic (`if-elif-else`)
This script dynamically evaluates an age variable and routes it through multi-stage age classification:

```python
age = 22

if age < 2:
    print("You are a baby!")
elif age < 4:
    print("You are a toddler!!")
elif age < 13:
    print("You are a kid!!!")
elif age < 20:
    print("You are a teenager")
elif age < 65:
    print("You are an adult")
else:
    print("You are an elder")
```

**Output:**
> `You are an adult`
